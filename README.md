# Discover Dhangadhi – City of Love

A fully responsive **static travel guide website** for Dhangadhi, Nepal, highlighting Tharu culture, attractions, hotels, food, and travel tips.

## Features
- Fully responsive design
- Hero section with gradient overlay
- Attraction, hotel, and food cards with hover effects
- Travel tips and contact form
- Smooth scroll and scroll-to-top button
- Fade-in animations on scroll

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts: Poppins & Open Sans
- Hosted easily on GitHub Pages or Netlify
